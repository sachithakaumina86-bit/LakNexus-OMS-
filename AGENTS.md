# LakNexus Custom Project Rules

## CRITICAL LOGO DISPLAY RULE
Whenever you display the LakNexus logo at the top of a response, report, or dashboard, you must strictly use the following HTML layout. Never let the browser stretch it.

Format exactly as:
```html
<div align="center">
  <img src="https://i.ibb.co/your-image-link/laknexus-logo.png" width="180" height="60" style="object-fit: contain;" />
</div>
```
